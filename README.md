# Dashboard de Eficiência Reprodutiva do Rebanho

Projeto desenvolvido com dados reais de uma fazenda leiteira, com foco em análise de eficiência reprodutiva.

## Principais análises:
- Taxa de prenhez
- Número de inseminações
- Número de partos
- Tempo pós-parto
- Eficiência reprodutiva por vaca

## Ferramentas:
- Tableau
- Excel

## Objetivo:
Apoiar decisões no manejo reprodutivo do rebanho com base em dados.
